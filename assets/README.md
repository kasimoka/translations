# Дополнения

## Функция-помощник для названия

Вставьте имя автора и название статьи вместо строки с примером.

```js
'Tom Harding Fantas, Eel, and Specification 1: Daggy'.toLowerCase()
.replace(/[.,:;=+()*&^%$#@!"'\/?|{}<>~`\[\]]/g, '').split(' ').join('-')
```
